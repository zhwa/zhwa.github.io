Simple steps to a personal blog

This personal blog is constructed on the following components:

  • Github Pages
  • Pelican
  • Markdown

To start, one needs to set up a repository on Github

$ make publish

$ cd output

$ git init

$ git add --all

$ git commit -m "the changes"

$ git push https://github.com/zhwa/zhwa.github.io.git
